# 기능 구현 사항

1. [ ] 사용자별 친구 관계파악 기능
2. [ ] 입력받은 user의 친구의 친구 목록을 얻는 기능
3. [ ] 사용자별 점수를 매기는 기능
4. [ ] 이름순으로 정렬해서 0이 아닌 친구를 추천하는 기능

# 예외사항

1. [ ] user의 길이가 0이하 30초과인경우
2. [ ] friends길이가 0이하 10000초과인경우
3. [ ] friends각 원소 배열 길이가 2가 아닌경우
4. [ ] id가 1이하, 30초과인경우
5. [ ] visitors가 10000초과인경우
6. [ ] id가 알파벳 소문자가 아닌 문자가 포함된 경우
